O bloco de envio de Vídeo permite enviar um vídeo para o contato. Você pode colar o URL de um arquivo **mp4** ou fazer upload do arquivo diretamente.
Também é possível adicionar uma legenda ao vídeo.

## Solução de Problemas

A mensagem não foi enviada ou não é possível reproduzir o vídeo?
- Verifique a URL do vídeo, ela deve ser o caminho do arquivo, por exemplo: `meusite.com/meuvideo.mp4`
- Verifique o formato do arquivo que deve ser **mp4**
