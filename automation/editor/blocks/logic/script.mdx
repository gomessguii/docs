O bloco "Script" o permite executar código Javascript.

<Info>Esse bloco não o permite criar um bloco visual personalizado</Info>

<Frame>
  <img src="/images/automation/blocks/logic/script.png" width="600" alt="Bloco de código" />
</Frame>

<Info>
As variáveis no script não são interpretadas, mas sim executadas. Portanto, elas devem ser tratadas como se fossem variáveis JavaScript reais.

Você precisa escrever `console.log({{Minha variavel}})` em vez de  `console.log("{{Minha variavel}}")`

</Info>

## Função `setVariable`

Se você quer definir um valor a uma variável com JavaScript, o [Bloco definir variável](./set-variable) é mais apropriado na maioria dos casos.

Porém, se você deseja definir variáveis dentro do seu bloco de script, você pode usar a função `setVariable` no seu script:

```js
if({{Minha variavel}} === 'valor1') {
  setVariable('Minha variavel', 'valor2')
} else {
  setVariable('Minha variavel', 'outro')
}
```
