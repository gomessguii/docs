## Converta texto para fala

Permite transformar um texto em fala usando vozes e modelos ElevenLabs. Esta ação cria automaticamente um link temporário para o arquivo de áudio. Este link pode ser usado em uma [mensagem de áudio](../outputs/audio) por exemplo.
