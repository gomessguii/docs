---
title: Menu de contexto
icon: bars
---

Você pode clicar com o botão direito em diferentes elementos para acionar um menu com diferentes ações.

Se você clicar com o botão direito em um grupo, verá um menu com as seguintes opções:

<Frame>
  <img
    src="/images/automation/group-context-menu.png"
    alt="Menu de contexto de grupo"
    className="rounded-lg"
  />
</Frame>

- **Duplicar** para criar uma duplicata do grupo com os mesmos blocos e configurações
- **Deletar** para remover o grupo

Se você clicar com o botão direito em um bloco, verá um menu com as seguintes opções:

- **Duplicar** para criar uma duplicata do bloco abaixo dele com as mesmas configurações
- **Deletar** para remover o bloco

Se você clicar com o botão direito numa linha, verá um menu com as seguintes opções:

- **Deletar** para remover a linha
