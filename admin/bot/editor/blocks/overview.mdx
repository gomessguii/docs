title: Visão Geral

Blocos são os elementos de construção de uma automação. Você pode encadear quaisquer blocos para criar fluxos de conversa complexos. Ao arrastar e soltar um bloco no fluxo da automação, por padrão ele será inserido em um grupo que pode ter um título. Os blocos estão sempre contidos em grupos.

Existem várias categorias de blocos para facilitar a localização do bloco certo para suas necessidades:

## Enviar

Blocos Enviar são usados para mostrar diversos tipos de informação ao usuário.

## Receber

Bloco Receber é usado para receber uma informação do usuário. Ele pára a conversa para que o usuário diga algo.

Blocos de lógica são usados para executar operações em segundo plano. Elas não visíveis aos usuários.
