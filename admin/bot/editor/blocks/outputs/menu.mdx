O bloco de envio Menu na verdade cria um grupo de blocos de exemplo de um menu. Ele adiciona:
- Um bloco de envio de Texto, com um texto de exemplo de menu com 3 alternativas.
- Um bloco Receber Resposta, salvando a entrada na variávei `escolha`
- Um bloco lógico Condição com três condicionais de exemplo

Esse atalho é útil para adicionar um menu ao fluxo sem precisar criá-lo do zero.
