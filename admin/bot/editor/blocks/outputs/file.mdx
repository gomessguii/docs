O bloco de envio de Arquivo permite enviar um arquivo ao contato. Você pode fazer upload do arquivo ou colar uma URL diretamente.

Também é possível enviar uma legenda com o arquivo.

<Note>
  Note que o envio de um arquivo pode demorar um pouco a depender do tamanho dele, principalmente na primeira vez que ele estiver sendo enviado.

  O tamanho do arquivo é delimitado pelo WhatsApp.
</Note>
