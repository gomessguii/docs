Tags são utilizadas para classificar seus contatos para serem utilizadas posteriormente nas automações.

## Criar uma Nova Tag

Para criar uma nova tag:
1. Clique em **Adicionar**
1. Defina um nome para ela
1. Selecione uma cor para identificá-la
1. Marque se quer que ela apareça ou não no Quadro (veja [Quadro](/customer-service/board)) e a ordem em que ela aparecerá no Quadro.
1. Clique em **Adicionar**

## Utilizar uma Tag

Para utilizar uma tag em um fluxo de automação, crie uma variável com o mesmo nome da Tag. Assim, o sistema preencherá a tag automaticamente. Veja o exemplo com uma Tag chamada "Perda":
