A página de contatos permite a criação de novos contatos, os lista e permite executar algumas ações com cada um.

## Adicionar Contato

1. Clique em Adicionar Contato, um formulário será apresentado.
1. Preencha as informações do seu contato. Somente **Nome** e **Número** são obrigatórios.
1. Opcionalmente, você pode utilizar Tags para categorizar os clientes. Veja [Tags](/tags).
1. Escolha se quer ou não bots ativados para esse contato. Útil para desativar bots em seus números de teste.
1. Opcionalmente, você pode utilizar os campos de **Informações Adicionais** para salvar informações importantes usadas no seus fluxos, para integrações com ferramentas externas, por exemplo. Esse tipo de informação é salvo com pares chave-valor.

## Listagem

Ao abrir a página, você encontrará uma tabela de contatos com as seguintes informações:
- **Id** com um botão para copiar o Id do contato para a área de transferência.
- **Foto de perfil** ou uma foto padrão caso o contato não tenha.
- **Nome do contato** ou o número caso ele não tenha um nome salvo.
- **Número de WhatsApp**.
- **Email** caso haja algum cadastrado.
- **Tag** caso haja alguma cadastrada.
- **Última mensagem** com a data de envio da última mensagem.
- **Status**

## Ações

Na listagem também há algumas ações que podem ser executadas para cada contato. São elas:
- **WhatsApp** - leva ao chat daquele contato na página de Atendimento.
- **Editar Contato** - abre uma caixa de diálogo com um formulário de edição para os dados do contato.
- **Bloquear Contato** - abre uma caixa de diálogo de confirmação para bloquear o contato no WhatsApp.
- **Deletar Contato** - abre uma caixa de diálogo de confirmação para deletar o contato do sistema.
