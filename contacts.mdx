A página de contatos permite a criação de novos contatos, os lista e permite executar algumas ações com cada um.

## Listagem

<Frame>
  <img
    src="/images/contacts/contacts.png"
    alt="Informações adicionais do contato"
    className="rounded-lg"
  />
</Frame>

Na listagem, ao clicar no botão ações em um contato, você pode:
- Abrir chat
- Editar o contato
- Bloquear ou desbloqueá-lo
- Excluí-lo
- Copiar seu ID

## Adicionar Contato

Para adicionar contatos você tem 3 opções. Clique no botão `Adicionar` para visualizá-las.

<Frame>
  <img
    src="/images/contacts/add-contact.png"
    alt="Informações adicionais do contato"
    className="rounded-lg"
    width={300}
  />
</Frame>

<Tabs>
  <Tab title="Adicionar Novo">
    Cria um novo contato a partir de informações inseridas manualmente.
    Para criar dessa forma, apenas selecione a opção e preencha os campos necessários.
  </Tab>
  <Tab title="Importar do WhatsApp">
    Importa os contatos já criados no WhatsApp
  </Tab>
  <Tab title="Importar do Excel">
    Importa os contatos a partir de uma planilha Excel criada a partir do modelo disponibilizado.
    Baixe o modelo [aqui](https://app.atendai.com/modelo.xlsx) ou diretamente na página de importação.
  </Tab>
</Tabs>

Você pode utilizar Tags para categorizar os clientes. Veja [Tags](/customer-service/tags).

Você pode utilizar os campos de **Informações Adicionais** para salvar informações importantes usadas no seus fluxos, para integrações com ferramentas externas, por exemplo. Esse tipo de informação é salvo com pares chave-valor.

<Note>Para configurar informações como Tags, Informações Adicionais ou UTMs em contatos importados, é necessário editá-los após a importação.</Note>

## Exportar Contatos

É possível exportar seus contatos para uma planilha do Excel. Para isso, clique no botão `Adicionar`e selecione a opção `Exportar para Excel`.

Os contatos serão exportados no mesmo modelo da planilha de importação.
